# materials/

Drop your study sources here and the coach will **read them and prefer them as
ground truth** when writing and grading questions:

- syllabus / learning outcomes
- your own notes and flashcards
- past papers **and their official solutions**

The grader cites which material an answer is grounded in. Anything it can't find
here is flagged **"general knowledge — verify against your source."**

## ⚠️ Copyright — read before adding anything

This project ships with **zero exam content**, and the `.gitignore` ignores
**everything in this folder except this README**. That is deliberate.

- **Do NOT commit copyrighted prep materials (TIA, ACTEX, ASM, official past-exam
  PDFs, etc.) to a PUBLIC repository.** That would infringe the publisher's
  copyright.
- If you want the coach to use copyrighted materials, **make your copy of this
  repository PRIVATE first** (GitHub → repo Settings → Change visibility →
  Private), then add them here.
- Materials you add are **yours and your responsibility**; the framework's MIT
  license does not cover them.

If your repo is public, the files you drop here stay on your machine and are
never pushed — only this README is tracked.
